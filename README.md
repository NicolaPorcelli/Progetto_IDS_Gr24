# Progetto di IDS - Registro Elettronico - Gruppo24 - Traccia_8 
## Descrizione
Progettazione di un sistema di registro elettronico di classe tramite software Visual Paradigm.
Inclusi nel progetto: file .vpp con i diagrammi della traccia analizzata, documentazione di analisi e specifica derivante dal file .vpp, implementazione in Java di alcuni dei casi d'uso seguendo le scelte di progettazione definite nella documentazione.
